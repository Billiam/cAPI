POST api​/v1​/ access_token

» Cakes » Cakes.com API v1

Generate access tokens from bakery credentials

Request

Example Request 

{
  "email": "user@example.com",
  "password": "battery horse staple"
}

Request Parameters

  • email

    The bakery email address

    • Required: yes
    • Type: string
    • Example: "user@example.com"
  • password

    The bakery password

    • Required: yes
    • Type: string
    • Example: "battery horse staple"

Response

Example Response 

{
  "access_token": {
    "code": "aaaaaaaaabbbbbbbbbbbccccccccccccddddddddd"
  }
}

Response Parameters

  • access_token
    • Required: no
    • code
      • Required: yes
      • Type: string
      • Example: "aaaaaaaaabbbbbbbbbbbccccccccccccddddddddd"
      • title: Access token code

Response Codes

Successful Response Codes

  • 201 Created

    Access token created successfully

Failure Response Codes

  • 401 Unauthorized

    Authentication failed
