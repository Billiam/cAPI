GET kiosk​/:kiosk_slug​/ status

» Cakes » kiosk

Get customer kiosk status

Request

Example Request 

{
  "kiosk_slug": "nerdery"
}

Request Parameters

  • kiosk_slug

    Customer slug. Part of the URL

    • Required: yes
    • Type: string
    • Example: "nerdery"

Response

Example Response 

{
  "name": "Nerdery",
  "cos_status": "active"
}

Response Parameters

  • name

    Customer name

    • Required: yes
    • Type: string
    • Example: "Nerdery"
  • cos_status

    Customer COS status

    • Required: yes
    • Type: string
    • Example: "active"

Response Codes

Successful Response Codes

  • 200

    A successful response
